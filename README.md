# Desafio Gama Pullrequest

### Cria uma BRANCH iniciada em "gama/"

#### Altere o arquivo README.MD e coloque suas impressões sobre este curso

##### Faça o commit da sua altetação

#### Execute o comando

`git push --set-upstream <remote> <branch>`
